Community Donations

Community donations (selected by either a port or the %percent% tag described above) go to a Stellite community donation pool which uses the funds to advance Stellite.

While we encourage donations to support the Stellite community, you are not required to donate to use this pool.

Note that donation levels are set per address, not per worker: if you have multiple workers with conflicting donation levels there is no guarantee of which one will be used.
